# Security policy

## Scope

This policy covers vulnerabilities **in hickok itself** — not the systems you
operate it against.

## Reporting

Report privately via GitHub Security Advisories ("Report a vulnerability" on the
repository's Security tab), or open a minimal issue if it is not sensitive.
Please include a description, affected version (`hickok --version`) and steps to
reproduce. Expect an initial response within a few days.

## Responsible use

hickok is offensive tooling. Only run it against systems you are permitted to
test. Misuse is the operator's responsibility, not the project's.
